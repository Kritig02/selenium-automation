# Astro Page Load Automation


Supported Browsers : 
1. Chrome (Browser + Headless)
2. Firefox (Gecko)

## Installation : 

```
    $ npm install
```

## Running Tests (Chrome Headless) : 

```
    $ npm test
```

### Testing on Chrome Browser : 

```
    $ npm run chrome
```

### Testing on Firefox Browser : 

```
    $ npm run firefox
```
